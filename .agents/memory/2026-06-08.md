- Brainstormed a source-neutral summarization boundary.
- Approved direction: loaders return `ContentDocument`; localized system and
  user prompts remain under `locales/`; source content is sent as `source.md`
  when supported with a single inline Markdown fallback for identifiable
  unsupported-file errors.
- Web page extraction is out of scope for this change, but future web loaders
  will use the same `ContentDocument` contract.
